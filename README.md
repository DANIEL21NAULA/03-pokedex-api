## Description

Proyect API POKEDEX

# Ejecutar en desarrollo
1. Clonar el repositorio
2. Ejecutar el siguiente comando

```bash
$ yarn install
```

3. tener instalado Nest CLI

```bash
$ npm i - g @nestjs/cli
```

4. Levantar la base de datos
```bash
$ docker-compose up -d
```


## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Stack usado
* MongoDB
* Nest
